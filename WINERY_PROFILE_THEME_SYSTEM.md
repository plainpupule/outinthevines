# Winery Profile Theme System v0.1

The profile system uses one shared data model and interaction pattern, but each winery receives a distinct visual theme.

## Shared structure
- Hero
- Vinny Verdict
- Quick Look icons
- Perfect For
- Maybe Skip If
- Vinny Scorecard
- Planning details
- Hours
- Events
- Reviews
- Andrew and Antonio picks
- Vinny Welcome Standard

## Winery-specific design expression

### Leoness — Estate
Elegant serif typography, muted rose and vineyard tones, crisp geometry, and restrained ornamentation.

### Akash — Modern
Clean sans serif typography, aqua accents, open spacing, rounded cards, and energetic asymmetry.

### Doffo — Moto
Slab serif typography, earthy leather tones, squared edges, heavier borders, and subtle mechanical textures.

### Europa Village — European
High-contrast serif typography, parchment tones, arches, ornamental spacing, and destination-resort romance.

### Wilson Creek — Celebration
Bright berry accents, rounded shapes, subtle confetti details, and high-energy group-oriented presentation.

## Design rule

The structure remains familiar enough that users can scan every profile quickly. The surface expression changes enough that each winery feels editorially distinct rather than templated.
